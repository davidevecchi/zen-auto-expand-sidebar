# Zen Auto Expand Sidebar

A Zen Browser mod that automatically expands the vertical sidebar on hover, providing a clean and efficient browsing experience.

## Features

- **Auto-expand sidebar** on hover with customizable delay and animation
- **Performance modes**: Fancy, Balanced, and Potato PC options
- **Customizable widths** for both collapsed and expanded states
- **Optional features**: fade sleeping tabs, hide workspace indicator, remove line separator
- **Essential tab alignment** to keep active tabs visible when collapsed
- **Full compatibility** with compact mode and single/multiple toolbar layouts

## Installation

1. Copy the `zen-auto-expand-sidebar` folder to your Zen Browser `chrome/sine-mods/` directory
2. Restart Zen Browser
3. Configure settings in the mods menu (gear icon)

## Important Notes

- **Turn the mod on/off** using fullscreen toggle (F11)
- **Use with single or multiple toolbar mode** - not compatible with collapsed toolbar mode
- **Works with compact mode** in all its forms, including "hide top bar only"

## Customization Options

Access these in the mods menu by clicking the gear icon:

### Performance Settings
- **Fancy**: Full animations and transitions for the best visual experience
- **Balanced**: Optimized transitions for good performance and visuals
- **Potato**: Minimal animations for maximum performance on low-end systems

### Sizing Options
- **Expanded sidebar width**: Controls the width when expanded (minimum 350px for right-side tabs)
- **Collapsed sidebar width**: Controls the width when minimized (default 45px)

### Animation Options
- **Animation duration**: Speed of the expanding animation (default: 150ms)
- **Expand delay**: Time before expanding after hover (default: 0ms)
- **Collapse delay**: Time before collapsing after unhover (default: 100ms)

### Visual Options
- **Fade sleeping tabs**: Slightly fade unloaded/sleeping tabs
- **Hide workspace indicator**: Remove the top workspace button
- **Remove line separator**: Remove the thin line under pinned tabs
- **Essential tab alignment**: Move selected essential tabs to first position when collapsed

## Troubleshooting

If the mod doesn't work:
1. Ensure you're not using collapsed toolbar mode
2. Try toggling fullscreen (F11) to enable/disable the mod
3. Check that the mod files are in the correct directory
4. Restart Zen Browser after making changes

## Compatibility

- ✅ Compact mode (all variants)
- ✅ Single toolbar mode
- ✅ Multiple toolbar mode
- ✅ Right-side vertical tabs
- ✅ Left-side vertical tabs
- ✅ Tab groups and folders
- ✅ Pinned tabs
- ✅ Essential tabs
